# Frechet derivative of matrix logarithm

Frechet derivative of matrix logarithm

## Usage

``` r
.frechet_log_apply(h_mat, prep)
```

## Arguments

- h_mat:

  Matrix to apply derivative

- prep:

  List from
  [`.frechet_log_prep`](https://jamesuanhoro.github.io/bayesianmasem/reference/dot-frechet_log_prep.md)

## Value

Matrix of same dimensions as h_mat
