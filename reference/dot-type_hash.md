# Type hash function

A function that swaps type from string to integer and vice-versa

## Usage

``` r
.type_hash(search_term = NULL, elaborate = FALSE)
```

## Arguments

- search_term:

  Integer or string or NULL

- elaborate:

  (LOGICAL) If TRUE, print full names, otherwise print abbreviations

## Value

If search_term is integer, returns string and vice-versa
