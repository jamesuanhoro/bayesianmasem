# Prepare for Frechet derivative of matrix logarithm

Prepare for Frechet derivative of matrix logarithm

## Usage

``` r
.frechet_log_prep(r_mat)
```

## Arguments

- r_mat:

  (matrix) Correlation matrix

## Value

A list with:

- q_vecs:

  Eigenvectors of `r_mat`

- f_mat:

  Matrix for Frechet derivative computation
