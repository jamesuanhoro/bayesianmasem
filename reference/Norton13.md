# Studies on the Hospital Anxiety and Depression Scale.

The data set includes 28 studies on 14 items measuring the Hospital
Anxiety and Depression Scale (HADS) Reported by Norton et al. (2013) ,
with data and documentation copied over from the metaSEM package (Cheung
2015) .

## Usage

``` r
Norton13
```

## Format

### `Norton13`

A list of data with the following structure:

- data:

  A list of 28 studies of correlation matrices. The variables are 14
  items (x1 to x14) measuring HADS

- n:

  A vector of sample sizes

- population:

  A vector of the population of the data

- group:

  A vector of classification into patients vs. non-patients based on
  population
