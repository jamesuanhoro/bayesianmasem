# Another dataset from the metaSEM package.

The data set includes 11 studies on 9 items measuring work-related
attitudes (Inter-university Consortium for Political and Social Research
1992) . Data and documentation copied over from the metaSEM package
(Cheung 2015) .

## Usage

``` r
issp89
```

## Format

### `issp89`

A list of data with the following structure:

- data:

  A list of 11 studies of covariance matrices

- n:

  A vector of sample sizes
